Profile Interpolation

This add-on allows the user to implement a profile interpolation lookup based on a CSV file.

A quick start document provides an overview of the addon capabilities. It is located at:

"C:\Users\Public\Documents\National Instruments\NI VeriStand (Year)\Custom Devices\Profile Interpolation\Windows\Quick Start Documentation\Profile Interpolation Quick Start.html"

Most of the addon sections contain additional built-in documentation.

NOTE: The typical use case for this add-on allows for interpolation for a defined set of position vs. load profiles.

What do you want to do?

Disable the Device

Right click on the custom device and select Disable.

Add an Interpolation File

To add an interpolation, select the file from the Interpolation File.
