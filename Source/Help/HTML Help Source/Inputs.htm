Inputs

Inputs groups all incoming channels.

  • Profile Select is a UI ring control that is automatically populated with the unique profile names in the interpolation table and is used to select the profile or subset of the table at run-time.


  • Measured channels should be mapped from other measurement channels and are typically Position and Load measurements. Names and units can be customized in the header row of the table.


  • Limit Enable is a UI control to enable limit checking. A True value will enable the limit check and report status on the Limit Violation Output channel.

Note: Limit Enable and some measurements channels are only created if Create Limit Channels? is set at initialization of the custom device.
