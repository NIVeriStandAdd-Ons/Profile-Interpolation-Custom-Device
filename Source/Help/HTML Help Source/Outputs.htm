Outputs

Outputs groups all outgoing channels.

  • Setpoint is the output of the interpolation function and should be mapped to the PID controller input.


  • Lower Limit is the interpolated lower limit based on the current position used to determine if the Limit Violation flag should be set.


  • Upper Limit is the interpolated upper limit based on the current position used to determine if the Limit Violation flag should be set.


  • Limit Violation is set to True when the measured input channel for load exceeds the specified and interpolated upper and lower limits.

Note: Upper Limit, Lower Limit and Limit Violation channels are only created if Create Limit Channels? is set at initialization of the custom device.
