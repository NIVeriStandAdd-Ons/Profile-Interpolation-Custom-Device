Status Channels

Status Channels lists channels that monitor the state and condition of various aspects of the system.

What do you want to do?

View status channel descriptions

To view a status channel description, select the status channel name in the configuration tree. The Status Channel Configuration page appears to the right of the configuration tree.

  • Profile Count is the number of unique profiles specified in the Interpolation File. Profile count can be used to check if the same number of profiles have been defined for multiple instances of the Profile Interpolation add-on.


  • Table Length is the total number of rows in the profile table including the header row. Table Length can be used to determine if points or profiles have been added.
